      todoist
      
          Pega tu token de API (Todoist → Configuración → Integraciones →
        Desarrollador). Cada tarea pendiente se envía una sola vez, con su proyecto,
        fecha, prioridad y #tags; las que completes en Todoist se marcan como
        hechas aquí, y las que crees en Todoist desde otro dispositivo se traen
        aquí, al sincronizar.
          
      
          
        
        
      
          
      
          
        
        
      
          
    
          

    
          
      escanear papel
      
          Apunta la cámara a una lista con casillas «- [ ]»: las líneas
        con casilla vacía se añadirán como to-dos del proyecto seleccionado
        (las marcadas «- [x]» se ignoran). El reconocimiento es local, sin internet.
